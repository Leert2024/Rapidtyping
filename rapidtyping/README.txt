A simple typing practise tool using text-based user interface.
一个简单的、使用字符用户界面的打字练习工具。

Programmed in C using Codeblocks.
使用C语言编写，使用Codeblocks。

To make sure that the programme can run properly, your cmd interface must have more than 100 columns. Run "mode" in cmd to see if that's so.
为确保程序正常运行，你的cmd界面的列数必须超过100。在cmd窗口中运行“mode”命令来检查是否如此。
